## 🧠 Geliştirme Felsefem & Odak Noktam

İzmir merkezli çalışmalarımda, akademik altyapımı gerçek dünya problemlerini çözen mimarilere dönüştürmeye odaklanıyorum. Sadece kod yazmakla kalmıyor, uygulamanın donanımla ve yapay zeka modelleriyle nasıl iletişim kurduğunu derinlemesine optimize ediyorum.

* 🔭 **Şu an üzerine yoğunlaştığım proje:** Dil modellerinin (LLM) kullanıcı konuşmalarını uzun vadeli hatırlamasını sağlayan açık kaynaklı bir bellek asistanı olan **HyprContext**.
* 📱 **Mobil vizyonum:** Bilgisayarlı Görü (Computer Vision) ve Doğal Dil İşleme (NLP) algoritmalarını Flutter ile uç cihazlara (Edge AI) entegre etmek. (Örn: AI destekli chatbotlar, sanal kıyafet deneme motorları).
* ⚙️ **Sistem yaklaşımım:** Arch Linux tabanlı geliştirme ortamlarında, Python ve R kullanarak veri süreçlerini yönetiyor, modelleri uçtan uca entegre ediyorum.
